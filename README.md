# Login App

A simple Java web application demonstrating login functionality using Servlets and JSP.

## Technologies Used
- Java
- Servlets
- Maven
- Tomcat
- HTML/CSS

## Setup Instructions
1. Clone the repository
2. Make sure you have Java 8+ and Maven installed
3. Run the application using:
```bash
mvn clean package tomcat7:run
```
4. Access the application at: http://localhost:8090/login-app/
